Dijkstra's Arena

A web-based game designed to educate students on Dijkstra's shortest path algorithm

Roger Wang, Anran Du, Jien Li, Xiaoyu Hu
Final project for CSCI3383

Created in Professor Lewis Tseng's Algorithms class, Dijkstra's Arena was the final project demonstrating our knowledge on the newly-learned Dijkstra's shortest path algorithm. The game includes both single-player and multiplayer components and is fun, challenging, and rewarding at the same time.
